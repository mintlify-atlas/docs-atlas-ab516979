WeKan is a free, open-source kanban board application built with Meteor. It gives you a real-time visual workspace to organize tasks, track progress, and collaborate with your team — all on infrastructure you control.

Used in most countries of the world, with the largest deployment serving 30,000 users, WeKan is a production-ready alternative to proprietary project management tools.

## Getting started

<Steps>
  <Step title="Choose your install method">
    WeKan runs on Docker, Snap, Sandstorm, and from source. [Docker](/installation/docker) is the fastest path for most teams.
  </Step>
  <Step title="Create your first board">
    After logging in, click **+ Add board** on the home screen. Give your board a name, set visibility (public or private), and pick a color.
  </Step>
  <Step title="Add lists and cards">
    Create lists to represent workflow stages (e.g., To Do, In Progress, Done). Add cards to lists and assign members, due dates, labels, and checklists.
  </Step>
  <Step title="Invite your team">
    Go to **Admin Panel → People** to invite users. Set roles (admin, normal, comment-only) per board as needed.
  </Step>
</Steps>

## Key features at a glance

| Feature | Description |
|---|---|
| Boards & Lists | Organize work into visual columns with drag-and-drop |
| Swimlanes | Horizontal rows that group cards by team, priority, or category |
| Custom Fields | Add structured data (text, numbers, dates, dropdowns) to cards |
| WIP Limits | Cap the number of cards per list to prevent bottlenecks |
| Checklists | Break cards into trackable sub-tasks |
| Gantt Charts | Visualize card timelines and dependencies |
| REST API | Full programmatic access to all WeKan resources |
| Webhooks | Push events to external services on any board action |
| Import/Export | Bring in data from Trello, Jira, Asana, or CSV |
| LDAP/OAuth2/SAML | Integrate with your existing identity provider |
