</Note>

## Get boards for a user

Retrieve all boards associated with a specific user.

| Method | Endpoint |
| ------ | -------- |
| `GET`  | `/api/users/:userId/boards` |

**Requires admin auth:** Yes

<ParamField path="userId" type="string" required>
  The ID of the user whose boards you want to retrieve.
</ParamField>

```bash
curl -H "Authorization: Bearer a6DM_gOPRwBdynfXaGBaiiEwTiAuigR_Fj_81QmNpnf" \
     http://localhost:3000/api/users/XQMZgynx9M79qTtQc/boards
```

## Create a board

Create a new board owned by a specified user.

| Method | Endpoint |
| ------ | -------- |
| `POST` | `/api/boards` |

**Requires admin auth:** Yes

### Request parameters

<ParamField body="title" type="string" required>
  The title of the board.
</ParamField>

<ParamField body="owner" type="string" required>
  The user ID of the board owner. This is the Wekan user ID, not a username or email address.
</ParamField>

<ParamField body="isAdmin" type="boolean" default="true">
  Whether the owner is an admin on this board.
</ParamField>

<ParamField body="isActive" type="boolean" default="true">
  Whether the owner is an active member of this board.
</ParamField>

<ParamField body="isNoComments" type="boolean" default="false">
  When `true`, the owner cannot add comments on this board.
</ParamField>

<ParamField body="isCommentOnly" type="boolean" default="false">
  When `true`, the owner can only comment and cannot make other changes.
</ParamField>

<ParamField body="permission" type="string" default="private">
  The visibility of the board. Accepted values: `"private"`, `"public"`.
</ParamField>

<ParamField body="color" type="string" default="belize">
  The background color of the board. Accepted values: `belize`, `nephritis`, `pomegranate`, `pumpkin`, `wisteria`, `moderatepink`, `strongcyan`, `limegreen`, `midnight`, `dark`, `relax`, `corteza`, `clearblue`, `natural`, `modern`, `moderndark`, `exodark`, `cleandark`, `cleanlight`.
</ParamField>

```bash
curl -H "Authorization: Bearer t7iYB86mXoLfP_XsMegxF41oKT7iiA9lDYiKVtXcctl" \
     -H "Content-type:application/json" \
     -X POST \
     http://localhost:3000/api/boards \
     -d '{"title":"Board title here","owner":"ABCDE12345","permission":"private","color":"nephritis"}'
```

### Response

<ResponseField name="_id" type="string">
  The ID of the newly created board.
</ResponseField>

```json
{
  "_id": "YRgy7Ku6uLFv2pYwZ"
}
```

## Add a board member

Add a user to a board with a specified role. The `action` field controls the operation. You can also use this endpoint to change an existing member's role.

| Method | Endpoint |
| ------ | -------- |
| `POST` | `/api/boards/:boardId/members/:memberId` |

**Requires admin auth:** Yes

<ParamField path="boardId" type="string" required>
  The ID of the board.
</ParamField>

<ParamField path="memberId" type="string" required>
  The user ID of the member to add.
</ParamField>

<ParamField body="action" type="string" required>
  Use `"add"` to add a new member, or omit to update an existing member's role.
</ParamField>

<ParamField body="isAdmin" type="boolean" default="false">
  Grant admin privileges on this board.
</ParamField>

<ParamField body="isNoComments" type="boolean" default="false">
  Restrict the member from commenting.
</ParamField>

<ParamField body="isCommentOnly" type="boolean" default="false">
  Restrict the member to comment-only access.
</ParamField>

<ParamField body="isWorker" type="boolean" default="false">
  Grant worker-level access. Workers can move cards, assign themselves, and comment, but cannot change settings or create/delete cards.
</ParamField>

**Add member with normal role:**

```bash
curl -H "Authorization: Bearer a6DM_gOPRwBdynfXaGBaiiEwTiAuigR_Fj_81QmNpnf" \
     -H "Content-type:application/json" \
     -X POST \
     http://localhost:3000/api/boards/YRgy7Ku6uLFv2pYwZ/members/ztKvBTzCqmyJ77on8/add \
     -d '{"action": "add", "isAdmin": "false", "isNoComments": "false", "isCommentOnly": "false", "isWorker": "false"}'
```

**Change member role to admin:**

```bash
curl -H "Authorization: Bearer a6DM_gOPRwBdynfXaGBaiiEwTiAuigR_Fj_81QmNpnf" \
     -H "Content-type:application/json" \
     -X POST \
     http://localhost:3000/api/boards/YRgy7Ku6uLFv2pYwZ/members/ztKvBTzCqmyJ77on8 \
     -d '{"isAdmin": "true", "isNoComments": "false", "isCommentOnly": "false", "isWorker": "false"}'
```

**Change member role to comment-only:**

```bash
curl -H "Authorization: Bearer a6DM_gOPRwBdynfXaGBaiiEwTiAuigR_Fj_81QmNpnf" \
     -H "Content-type:application/json" \
     -X POST \
     http://localhost:3000/api/boards/YRgy7Ku6uLFv2pYwZ/members/ztKvBTzCqmyJ77on8 \
     -d '{"isAdmin": "false", "isNoComments": "false", "isCommentOnly": "true", "isWorker": "false"}'
```

## Take ownership of all boards

Allows an admin to take ownership of all boards belonging to a given user â€” including archived boards where that user was an admin.

| Method | Endpoint |
| ------ | -------- |
| `PUT`  | `/api/users/:id` |

**Requires admin auth:** Yes

<ParamField path="id" type="string" required>
  The user ID whose boards will be transferred to the calling admin.
</ParamField>

<ParamField body="action" type="string" required>
  Must be `"takeOwnership"`.
</ParamField>

```bash
curl -H "Authorization: Bearer t7iYB86mXoLfP_XsMegxF41oKT7iiA9lDYiKVtXcctl" \
     -H "Content-type:application/json" \
     -X PUT \
     http://localhost:3000/api/users/ztKvBTzCqmyJ77on8 \
     -d '{ "action": "takeOwnership" }'
```

<Warning>
  This action is irreversible. All boards where the target user was an admin will be transferred to the authenticated admin making the request.
</Warning>
